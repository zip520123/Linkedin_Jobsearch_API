How it works

  1. Enter your GitHub PAT (Settings -> Developer Settings -> Tokens).
  2. Modify the JSON config (Keywords, Locations).
  3. Click "Run Search Workflow".
  4. Wait ~1-2 minutes for the Action to finish.
  5. Click "Refresh Data" to see new results.
